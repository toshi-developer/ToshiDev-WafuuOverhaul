# ToshiDev Wafuu Overhaul

7 Days to Die v1.0+ 向けの和風オーバーホール MOD です。

ゾンビアポカリプスの舞台を日本に置き換え、和の文化・食・武器・建築・敵を統合した大型 MOD を目指しています。

## インストール

1. [Releases](https://github.com/toshi-developer/ToshiDev-WafuuOverhaul/releases) から最新の ZIP をダウンロード
2. 解凍して `ToshiDev-WafuuOverhaul` フォルダごと `7 Days to Die/Mods/` に配置
3. ゲームを起動

```
7 Days to Die/
└── Mods/
    └── ToshiDev-WafuuOverhaul/
        ├── ModInfo.xml
        ├── Config/
        └── UIAtlases/
```

## 現在の内容（Phase 1: 和食拡張）

- 和食レシピ 70 種以上（調味料・主食・おかず・丼もの・汁物・甘味・飲み物）
- 和風クラフトステーション（かまど・囲炉裏・厨房セット）
- パーク「和食の匠」（専用スキルツリー）
- 日英ローカライズ対応
