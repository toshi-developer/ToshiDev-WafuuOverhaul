# ToshiDev Wafuu Overhaul

7 Days to Die v1.0+ 向けの和風オーバーホール MOD です。

亡者（ゾンビ）蔓延る終末の舞台を日本に置き換え、和の文化・食・武器・建築・敵を統合した大型 MOD を目指しています。

## インストール

1. [Releases](https://github.com/toshi-developer/ToshiDev-WafuuOverhaul/releases) から最新の ZIP をダウンロード
2. 解凍して `ToshiDev-WafuuOverhaul` フォルダごと `7 Days to Die/Mods/` に配置
3. ゲームを起動

```
7 Days to Die/
└── Mods/
    └── ToshiDev-WafuuOverhaul/
        ├── ModInfo.xml
        ├── Config/
        └── UIAtlases/
```

## 現在の内容

### Phase 1: 和食拡張
- 和食アイテム 70 種以上（調味料・主食・おかず・丼もの・汁物・甘味・飲み物）
- 和風調理設備 3 種（かまど・囲炉裏・厨房一式）
- 技能「和食の匠」（専用技能樹）
- 独自の効果 21 種（寒冷耐性・闘気・滋養・韋駄天・集中 等）
- 商人・キッチン・冷蔵庫等に和食素材の出現追加

### Phase 2: 和風武器・防具
- 近接武器 9 種（木刀・竹槍・打刀・薙刀・太刀・十文字槍・金棒・鎖鎌・妖刀）
- 遠距離武器 3 種（和弓・半弓・火縄銃）
- 矢 4 種（竹矢・火矢・毒矢・破甲矢）
- 投擲武器 3 種（手裏剣・くない・煙玉）
- 防具 4 系統 16 部位（足軽・侍・忍・大将）
- 鍛冶素材（玉鋼・刀身）
- 技能 5 種（剣術・槍術・弓術・忍術・鍛冶）

### Phase 3: 和風建築
- 壁材 5 種（土壁・漆喰壁・板壁・石垣・なまこ壁）
- 床材 3 種（畳・板張り・石畳）
- 屋根 2 種（黒瓦・赤瓦）
- 建具 4 種（障子・襖・格子戸・引き戸）
- 柱 3 種（丸柱・角柱・鳥居柱）
- 家具 5 種（ちゃぶ台・座布団・布団・箪笥・屏風）
- 照明 3 種（行灯・提灯・石灯籠）
- 装飾 3 種（掛け軸・神棚・盆栽）
- 防御 4 種（竹垣・竹槍防柵・城門・矢狭間）

### Phase 4: 和風妖怪
- 妖怪亡者 7 種（腐武者・怨霊・餓鬼・鬼・河童・化け狸・般若）
- 序盤〜終盤・水辺の出現グループ 4 種
- バニラの出現グループにも和風妖怪を追加
- 妖怪素材ドロップ 5 種（鬼の角・怨霊玉・河童の皿・狸の毛皮・般若の面）

### Phase 4.1: 和風 UI
- 和風配色 25 色（漆黒・金・朱・温白・渋藍・琥珀 等）
- HUD 全面和風リデザイン（ステータスバー・コンパス・ツールベルト・ターゲットバー・車両バー）
- 画面各所のパネル・枠線の和風化
- 和風読込画面の助言 37 種
- メニュー背景を和風画像（鳥居・月・桜）に差し替え（UIAtlas スプライト方式）
- インベントリ拡張（45 → 72 スロット）

### Phase 7: 和漢薬・医療品
- 薬草（素材）: 食料コンテナ・医療コンテナからルート入手
- 薬草湯: HP 緩回復（60 秒、+1.5 HP/s）
- 傷薬: 止血 + HP 回復
- 解毒散: 感染除去 + 免疫強化
- お守り: 全耐性強化（感染・物理・寒暑 120 秒）
- 清め塩: 感染予防（300 秒）
- 全品を医療コンテナ・商人（千代）からも入手可能

### ローカライズ拡張
- 乗り物の和名化（足踏み車・鉄馬・飛行凧・荷車）
- バイオーム名の和名化（焼け野原・荒野・里山・雪国・荒廃地）
- 依頼（クエスト）用語の和訳（討伐・回収・掃討・修復・試練・段）

### 商人の品揃え和風化
- 丈助（万屋）: 基本食料・木刀・薬草
- 六兵衛（武具商）: 全和武器・防具・弾薬・玉鋼
- 文左衛門（食料商）: 全和食・飲料・調味料・食材
- 兵庫（建材商）: 建築素材（木材・石材・鉄・鋼）
- 千代（薬種商）: 和漢薬・薬草・妖怪素材
