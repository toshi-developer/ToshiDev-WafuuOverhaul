# ToshiDev Wafuu Overhaul

7 Days to Die v1.0+ 向けの和風オーバーホール MOD です。

亡者（ゾンビ）蔓延る終末の舞台を日本に置き換え、和の文化・食・武器・建築・敵を統合した大型 MOD を目指しています。

## インストール

1. [Releases](https://github.com/toshi-developer/ToshiDev-WafuuOverhaul/releases) から最新の ZIP をダウンロード
2. 解凍して `ToshiDev-WafuuOverhaul` フォルダごと `7 Days to Die/Mods/` に配置
3. ゲームを起動

```
7 Days to Die/
└── Mods/
    └── ToshiDev-WafuuOverhaul/
        ├── ModInfo.xml
        ├── Config/
        └── UIAtlases/
```

## 現在の内容

### Phase 1: 和食拡張
- 和食アイテム 70 種以上（調味料・主食・おかず・丼もの・汁物・甘味・飲み物）
- 和風調理設備 3 種（かまど・囲炉裏・厨房一式）
- 技能「和食の匠」（専用技能樹）
- 独自の効果 21 種（寒冷耐性・闘気・滋養・韋駄天・集中 等）
- 商人・キッチン・冷蔵庫等に和食素材の出現追加

### Phase 2: 和風武器・防具
- 近接武器 9 種（木刀・竹槍・打刀・薙刀・太刀・十文字槍・金棒・鎖鎌・妖刀）
- 遠距離武器 3 種（和弓・半弓・火縄銃）
- 矢 4 種（竹矢・火矢・毒矢・破甲矢）
- 投擲武器 3 種（手裏剣・くない・煙玉）
- 防具 4 系統 16 部位（足軽・侍・忍・大将）
- 鍛冶素材（玉鋼・刀身）
- 技能 5 種（剣術・槍術・弓術・忍術・鍛冶）

### Phase 3: 和風建築
- 壁材 5 種（土壁・漆喰壁・板壁・石垣・なまこ壁）
- 床材 3 種（畳・板張り・石畳）
- 屋根 2 種（黒瓦・赤瓦）
- 建具 4 種（障子・襖・格子戸・引き戸）
- 柱 3 種（丸柱・角柱・鳥居柱）
- 家具 5 種（ちゃぶ台・座布団・布団・箪笥・屏風）
- 照明 3 種（行灯・提灯・石灯籠）
- 装飾 3 種（掛け軸・神棚・盆栽）
- 防御 4 種（竹垣・竹槍防柵・城門・矢狭間）

### Phase 4: 和風妖怪
- 妖怪亡者 7 種（腐武者・怨霊・餓鬼・鬼・河童・化け狸・般若）
- 序盤〜終盤・水辺の出現グループ 4 種
- バニラの出現グループにも和風妖怪を追加

### Phase 4.1: 和風 UI
- 和風配色 22 色（漆黒・金・朱・温白 等）
- HUD 色変更（体力・気力・食料・水分・経験値）
- 画面各所のパネル・枠線の和風化
- 和風読込画面の助言 6 種
